You can process the import in one of two ways:

Sequential import:

This will process each resource individually and provide real-time monitoring. However, if you navigate away from this page the import will stop and you will have to restart from where you left off. This method is better for large imports that might cause the server to timeout.

Begin Sequential Import

Batch import:

This will attempt to process the entire import file in one go. You should still see the monitor above update automatically every few seconds, but it will not update after every resource. Using this option you can navigation away from this page and the import should continue.

Begin Batch Import
